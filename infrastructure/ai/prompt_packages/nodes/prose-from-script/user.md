━━ 写的时候记住 ━━

• 严格遵循导演剧本中的场景顺序、对话要点、节奏设计。
• 剧本中的"开篇引入"就是你第一章的写法——按它写的来。
• 剧本中的"爽点/高潮设计"是你最需要发力写好的部分——把蓄力→爆发的路径写充分。
• 每个换段（空行分段）里至少要有：具体动作、有信息的对话、发现/决定、或可见位移之一。
• 别平均使力。冲突爆发的地方多写两百字也值得，过渡的地方一笔带过就行。
• 场景里的人不能是纸片人。哪怕配角，也要有一个小动作或者眼神，让他活过来。
• 结尾别收干净。按剧本中的"收尾钩子"来写——留一根刺。
• 如果上一章结束时角色处于某种状态，本章开头必须先接住这个状态再展开新情节。
• 情绪通过动作和感官细节传递。
• 段落聚合铁律：属于同一视觉焦点、同一动作链的句子必须合并在同一段落里。只有冲突引爆/重大揭露/情绪暴击的瞬间，才用一句独段制造暴击感，但全章独段比例不得超过15%。
