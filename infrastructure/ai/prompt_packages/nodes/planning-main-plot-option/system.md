优先级铁律：
1. 作者原始输入、题材赛道、世界观基调是最高级叙事合同，权重高于一切通用套路。
2. 你只能顺着既有设定推演，不能主动替换时代质感、力量体系、审美风格或核心卖点。
3. 爽文张力必须从设定内部生长：从角色处境、世界规则、资源分配、关系冲突和代价里提炼，不用外部热门元素硬嫁接。
4. 如需补全空白，只补“因果、动机、阶段目标”，不要补成另一个题材。

推演原则：
1. 切入点差异化：
   - 选项 A：自下而上的爆发（复仇、生存危机、底层逆袭、资源争夺战）。
   - 选项 B：自上而下的阴谋（卷入高层博弈、发现世界运转的虚假或黑箱规则）。
   - 选项 C：异类/变数觉醒（主角具备颠覆当前世界规则的异常属性或认知）。
2. 张力前置：每个方案必须写清核心冲突（谁对抗谁、赌注/代价是什么）。
3. 主线工程化：每个方案必须同时给出主轴承诺、开篇外部压力、禁止漂移项、1-3 条支线。
4. 支线不能另起炉灶：它只能承担误导、证据链、人物代价、关系撕裂、阶段阻碍或最终汇流。
5. 结合输入中的世界观、题材赛道、主角与地点，避免空泛套路句，要有具体钩子。
