【输出格式】
请先给出简短的"叙事策略推演"（一句话说明你将如何运用上述法则），然后严格按以下六个模块输出，每个模块的生成都要能体现法则的运用：

1. **开篇引入**
   - 本章第一段的写法（从什么画面/动作/对话切入）
   - 如何接住前章的悬念或情绪惯性
   - 前三句话的节奏设计

2. **场景描写要点**
   - 本章涉及的主要场景（1-3个）
   - 每个场景的环境如何反映角色心境（视域锚定）
   - 感官细节的使用策略（温度/光线/声音/触感/气味）

3. **对话设计要点**
   - 本章关键对话（谁与谁、聊什么、表面/底层分别是什么）
   - 每段对话的信息释放量（信息锥度）
   - 对抗感设计（情感压强差的具体体现）

4. **情节推进节奏**
   - 本章的节奏曲线（哪里慢写、哪里快进）
   - 情节点的时间分布（时间膨胀/压缩的具体位置）
   - 势能变化轨迹（蓄势→引爆→余韵的节奏设计）

5. **角色行为细节**
   - POV角色的情绪弧线（从什么状态到另什么状态）
   - 配角的功能性出场（谁、在哪个节点、推动什么）
   - 每个角色的标志性动作/反应设计

6. **爽点/高潮设计**
   - 本章的核心爽点是什么（实力爆发/身份反转/打脸/关系突破等）
   - 蓄力→爆发的具体路径
   - 旁观者反应的层次设计
   - 收尾钩子（留给下一章的悬念/期待）

请确保每个模块的内容具体、可执行，不要空泛地写"要有冲突"、"要有张力"，而是明确指出冲突的具体形态、张力的具体来源。剧本是给"下一阶段的小说作家AI"看的，所以细节越具体越好。
