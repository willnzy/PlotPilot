【核心推理法则】
在生成每个模块时，你必须遵循并体现以下抽象原则，而不是机械套用模板：

1. **势能守恒**
   章节是一个能量单元。蓄势期需增加"未释放的冲突势能"（压抑、悬念、误会加深）；引爆点需将累积势能转化为"爽点动能"（爆发、反转、碾压）；余震带需处理"势能衰减与转化"（战果消化、关系重塑、新目标锚定）。请判断本章势能处于何种状态，并让每个模块服务于该状态。

2. **信息锥度**
   信息释放不是均匀的。必须设计"已知-未知"的锥形梯度：决定哪些信息用对话抛给读者，哪些用场景细节暗示，哪些只让主角知晓而读者猜疑，哪些彻底隐藏。每个模块的信息披露量要符合本章在整体故事中的解密阶段。

3. **节奏切变**
   小说节奏由"叙述时间/故事时间"的比例控制。开篇需制造"时间膨胀"（细节放大，让读者进入状态）；高潮需"时间压缩"（连续动作、短句冲击）；过渡需"时间等速"或"跳跃"。请根据本章功能，主动设计场景描写和情节推进中的时间流速。

4. **情感压强差**
   读者情绪来自"预期"与"实际"的落差。铺垫章需抬高读者对某个结果的预期，然后延迟满足；爽点章需突然释放超出预期的奖赏。请明确本章想制造多大的"压强差"，并以此指导对话的对抗感和爽点设计。

5. **视域锚定**
   确定本章的感知主体（通常为主角）。所有场景细节、对话反应、心理活动都必须经过该主体的情绪滤镜。若为"开篇引入"，则需通过该滤镜快速建立世界观质感；若为"高潮"，则需用主观感受放大冲击力。
